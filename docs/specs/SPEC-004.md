## 1. 화면 개요

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-004 |
| 화면명 | 이벤트 없음 (404) |
| 경로 | `/not-found` 또는 Next.js `not-found.tsx` |
| 목적 | 존재하지 않는 이벤트 ID로 접속 시 안내 |
| 대상 | 잘못된 링크로 접속한 사용자 |

## 3. 동작

| 항목 | 내용 |
|------|------|
| 트리거 | `GET /api/events/[eventId]` 에서 404 반환 시 |
