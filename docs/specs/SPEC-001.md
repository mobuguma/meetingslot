## 2. 레이아웃 (반응형)

> 반응형 기준점: **600px** (Tailwind 커스텀 브레이크포인트 `sm600` 추가 필요)
> `tailwind.config.ts` 의 `screens` 에 `'sm600': '600px'` 추가

### 넓은 화면 (600px 이상) — 좌우 2컬럼

| 영역 | 위치 | 내용 |
|------|------|------|
| 좌측 (flex-1) | 왼쪽 | 이벤트 이름, 시간 범위, 날짜 선택, 타임존, CTA 버튼 |
| 우측 (sm600:w-96) | 오른쪽 | 가용 시간 테이블 미리보기 (SCR-002 스타일) |

```
[이벤트 이름    ] │ ┌──────────────────┐
                  │ │  가용 시간 미리보기 │
[시작▼] ~ [종료▼] │ │                   │
┌────────────┐   │ │  (선택된 날짜/시간  │
│  달력       │   │ │   기반 테이블)     │
│ (다중선택)  │   │ │                   │
└────────────┘   │ └──────────────────┘
[타임존       ▼] │
[ 이벤트 만들기 ] │
```

### 모바일 (< 600px) — 상하 단일 컬럼

| 순서 | 내용 |
|------|------|
| 1 | 이벤트 이름 입력 |
| 2 | 시간 범위 드롭다운 |
| 3 | 날짜 선택 달력 (full-width) |
| 4 | 타임존 선택 |
| 5 | 가용 시간 테이블 미리보기 (full-width) |
| 6 | 이벤트 만들기 버튼 (full-width) |

**Tailwind 구조**
```
<div class="flex flex-col sm600:flex-row gap-8">
  <div class="flex-1"> <!-- 폼 영역: 이름, 시간범위, 달력, 타임존, CTA --> </div>
  <div class="sm600:w-96 flex-shrink-0"> <!-- 가용 시간 미리보기 --> </div>
</div>
```

## 3. 입력 항목 명세

| 항목 | 타입 | 필수 | 유효성 | 비고 |
|------|------|------|--------|------|
| 이벤트 이름 | Text Input | ✓ | 1~50자 | placeholder: "예) 팀 주간 회의" |
| 시작 시간 | Select | ✓ | 종료 시간보다 이전 | 00:00~23:30, **30분 단위**, 기본값 09:00; 드롭다운 박스 좌우 내부 패딩 균등 적용 (화살표 아이콘 포함 영역까지 일관된 padding-x 유지) |
| 종료 시간 | Select | ✓ | 시작 시간보다 이후 | 00:30~24:00, **30분 단위**, 기본값 18:00; 동일 스타일 적용 |
| 날짜 선택 | Calendar (다중) | ✓ | 1일 이상, 최대 14일 | 오늘 이후 날짜만, 클릭앤드래그 선택/해제, '모두 해제' 버튼 (항상 노출) |
| 타임존 | Searchable Select | ✓ | - | 전체 IANA 타임존 목록 표시 (**"Area/Location (GMT+N)"** 형식); 텍스트 입력으로 실시간 필터링; 기본값: 브라우저 타임존 자동 감지; 서버 저장 시 UTC 변환 |

## 4. 동작 상세

### 4.1 이벤트 생성

| 항목 | 내용 |
|------|------|
| 트리거 | "이벤트 만들기" 버튼 클릭 |
| 유효성 검사 | 이름 공백, 날짜 미선택 → 인라인 에러 표시 |
| 동작 | `POST /api/events` 호출 → 고유 eventId 생성 (nanoid 방식) |
| 성공 피드백 | 생성 완료 화면으로 전환 (→ 4.5 이벤트 생성 완료 화면 참조) |
| 실패 피드백 | 토스트 에러 메시지 (중앙 정렬) |

#### API 명세: `POST /api/events`

```typescript
// Request Body
{
  title: string;      // 이벤트 이름 (1~50자)
  dates: string[];    // 선택된 날짜 배열 (YYYY-MM-DD 형식)
  startTime: string;  // 시작 시간 (HH:00 형식, UTC 기준)
  endTime: string;    // 종료 시간 (HH:00 형식, UTC 기준)
  timezone: string;   // IANA 타임존 문자열 (예: "Asia/Seoul")
}

// Response
{
  eventId: string;    // 생성된 고유 ID (8자 hex, crypto.randomBytes(4) 기반)
  url: string;        // 공유 URL (예: "https://meetslot.com/{eventId}")
}
```

> 슬롯 데이터 형식: `"20260320T090000"` (YYYYMMDDTHHmmss, UTC 기준)

### 4.2 시간 범위 보정

| 항목 | 내용 |
|------|------|
| 트리거 | 종료 시간을 시작 시간과 같거나 이전 값으로 변경 시 즉시 발생 |
| 동작 1 | 토스트 에러 메시지 노출: "종료 시간은 시작 시간보다 늦어야 합니다" (중앙 정렬) |
| 동작 2 | 종료 시간을 자동 보정: `시작 시간 + 1시간` 으로 설정 (30분 단위 유지; 최대 24:00으로 제한) |
| 미리보기 sync | 시간 변경 즉시 우측 가용 시간 테이블에 반영 |

### 4.3 날짜 선택

| 항목 | 내용 |
|------|------|
| 단일 클릭 | 날짜 선택/해제 토글 |
| 클릭앤드래그 | 드래그 시작 시점의 날짜 상태(선택/해제)를 기준으로 드래그 범위 일괄 적용 |
| 최대 선택 | 14일 초과 시 추가 선택 불가, 토스트 안내: "최대 14일까지 선택할 수 있습니다" (중앙 정렬) |
| 모두 해제 | **날짜 선택 여부와 무관하게 항상** 달력 하단에 'N일 선택됨' + '모두 해제' 버튼 노출; 클릭 시 전체 해제 |
| 미리보기 sync | 날짜 변경 즉시 우측 가용 시간 테이블에 반영 |

### 4.4 링크 복사

| 항목 | 내용 |
|------|------|
| 트리거 | "링크 복사" 버튼 클릭 |
| 동작 | `navigator.clipboard.writeText(url)` |
| 피드백 | 버튼 텍스트 "복사됨 ✓" 로 2초간 변경 후 복원 |

### 4.5 이벤트 생성 완료 화면

이벤트 생성(`POST /api/events`) 성공 시 폼을 완료 화면으로 교체하여 표시한다.

#### 레이아웃

```
┌─────────────────────────────────────────┐
│                                         │
│   이벤트가 생성되었습니다!               │  ← 완료 타이틀 (헤더 타이틀 '새 이벤트 만들기'는 미노출)
│                                         │
│   [이벤트명]                            │  ← 생성된 이벤트 이름 표시
│                                         │
│   공유 링크                             │  ← 섹션 레이블
│   ┌───────────────────────────────┐    │
│   │  https://meetslot.com/abc123  │    │  ← 생성된 URL (읽기 전용)
│   └───────────────────────────────┘    │
│                                         │
│   [링크 복사]   [이벤트 참여 →]         │  ← 버튼 2개 가로 배치
│                                         │
└─────────────────────────────────────────┘
```

#### 완료 화면 상세 명세

| 항목 | 내용 |
|------|------|
| 헤더 타이틀 | **미노출** ('새 이벤트 만들기' 텍스트 숨김 처리) |
| 완료 타이틀 | "이벤트가 생성되었습니다!" |
| 이벤트명 | 생성된 이벤트 이름 텍스트 표시 |
| 공유 링크 | 생성된 URL 전체 표시 (읽기 전용 텍스트 또는 input[readonly]) |
| 버튼 배치 | '링크 복사' 버튼 좌측, '이벤트 참여' 버튼 우측으로 가로 배치 |

#### 버튼 명세

**링크 복사 버튼**

| 항목 | 내용 |
|------|------|
| 텍스트 | "링크 복사" |
| 동작 | `navigator.clipboard.writeText(url)` |
| 피드백 | 버튼 텍스트 "복사됨 ✓" 로 2초간 변경 후 복원 |
| 실패 시 | URL 텍스트 선택 처리로 fallback |

**이벤트 참여 버튼**

| 항목 | 내용 |
|------|------|
| 텍스트 | "이벤트 참여" |
| 동작 | `window.open(url, '_blank')` — 새 탭에서 생성된 이벤트 참여 화면(SCR-002) 열기 |
| 위치 | '링크 복사' 버튼 바로 우측 |

## 5. 컴포넌트 명세

### EventForm

| 항목 | 내용 |
|------|------|
| 역할 | 이벤트 생성 폼 전체 관리 (상태 + 제출) |
| 상태 | `title`, `dates`, `startTime`, `endTime`, `timezone`, `isLoading`, `createdUrl` |
| 제출 | `handleSubmit` → POST /api/events |

### DatePicker

| 항목 | 내용 |
|------|------|
| 역할 | 달력 UI에서 날짜 다중 선택 |
| 입력 | `selectedDates: string[]`, `onChange: (dates: string[]) => void` |
| 동작 | 날짜 클릭 시 선택/해제 토글, 클릭앤드래그로 다중 선택/해제, 오늘 이전 날짜 비활성화 |
| 모두 해제 버튼 | **항상** 달력 하단에 'N일 선택됨' + '모두 해제' 노출 (날짜 미선택 시 "0일 선택됨" 표시); 클릭 시 전체 해제 |
| 네비게이션 | 이전/다음 달 이동 버튼 |

### TimezoneSelector

| 항목 | 내용 |
|------|------|
| 역할 | IANA 타임존 목록에서 검색 및 선택 |
| 표기 형식 | `"Area/Location (GMT+N)"` — 예: "Asia/Seoul (GMT+9)", "America/New_York (GMT-5)" |
| 검색 | 드롭다운 열면 검색창 노출; 입력 텍스트가 포함된 타임존만 실시간 필터링 (대소문자 무시) |
| 기본값 | 브라우저 타임존 자동 감지 (`Intl.DateTimeFormat().resolvedOptions().timeZone`) |
| 외부 클릭 | 드롭다운 외부 클릭 시 자동 닫기 |

### AvailabilityPreview

| 항목 | 내용 |
|------|------|
| 역할 | 선택된 날짜/시간 범위를 바탕으로 가용 시간 테이블을 즉시 미리보기 |
| 입력 | `dates: string[]`, `startTime: string`, `endTime: string` |
| 동작 | `dates` 또는 `startTime`/`endTime` 변경 시 즉시 테이블 업데이트 (SCR-002 스타일) |
| 빈 상태 | 날짜 미선택 시 "날짜와 시간을 선택하면 미리보기가 표시됩니다" 안내 텍스트 |

## 6. Error Handling

| # | 예외 상황 | 동작 | 토스트 위치 |
|---|---------|------|------------|
| 1 | 이벤트 이름 미입력 | "이벤트 이름을 입력해 주세요" 인라인 에러 | - |
| 2 | 날짜 미선택 상태로 제출 | "날짜를 하나 이상 선택해 주세요" 인라인 에러 | - |
| 3 | 종료 시간 ≤ 시작 시간 | "종료 시간은 시작 시간보다 늦어야 합니다" 토스트 + 종료 시간 자동 보정(시작+1h) | 중앙 정렬 |
| 4 | 날짜 14일 초과 선택 | "최대 14일까지 선택할 수 있습니다" 토스트, 추가 선택 차단 | 중앙 정렬 |
| 5 | API 실패 | "이벤트 생성에 실패했습니다. 다시 시도해 주세요" 토스트 | 중앙 정렬 |
| 6 | 클립보드 복사 실패 | URL 텍스트 선택 처리로 fallback | - |
